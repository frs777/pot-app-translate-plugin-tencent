# Pot-App Wtyczka Tłumaczeniowa

## Sposób użycia

1. Pobierz wtyczkę dla swojej platformy, otrzymasz plik `.potext`
2. Otwórz Pot -> Preferencje -> Ustawienia usług -> Tłumaczenie -> Dodaj zewnętrzną wtyczkę -> Zainstaluj zewnętrzną wtyczkę
3. Wybierz pobrany plik `.potext`, instalacja powiodła się
4. Dodaj wtyczkę do listy usług, gotowe!
